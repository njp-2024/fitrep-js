# FITREP Calculator

A web-based fitness report calculator application.

## Project Status

- UI: ~90% complete
- Logic: ~60% complete (untested)

## Structure

```
fitrep-calculator/
├── index.html          # Main HTML file
├── style.css           # Main stylesheet
├── js/
│   ├── app.js          # Application entry point
│   ├── Config.js       # Configuration settings
│   ├── models/         # Data models (Profile, Report)
│   ├── services/       # Business logic (Calculator, Validation, PrecisionService)
│   ├── state/          # State management (Store)
│   └── ui/             # UI components (Navigation, Sidebar, Pages)
└── css/                # Additional stylesheets
```

## Getting Started

1. Clone the repository
2. Open `index.html` in a web browser

## Technology Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

## Development

This is a vanilla JavaScript project with no build tools or dependencies. Simply edit files and refresh the browser to see changes.
